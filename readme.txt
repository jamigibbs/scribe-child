== Description ==

This child theme is for the Scribe WordPress theme by Jami Gibbs. The parent theme is available on the WordPress theme directory:
http://wordpress.org/themes/scribe

== Installation ==

1. Install the parent theme, Scribe.
2. Install this child theme and activate it.
3. Continue with your theme setup and customization within the child theme.

More information about child themes can be found on WordPress.org:
https://codex.wordpress.org/Child_Themes
